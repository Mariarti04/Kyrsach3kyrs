# Информационная система управления медицинской организацией

Курсовой проект по дисциплине "Базы данных"

## Описание проекта

Разработано веб-приложение для управления медицинской организацией (поликлиника/больница) с использованием Django и Django REST Framework.

### Основные возможности:

1. **Управление пациентами** - ведение электронных медицинских карт
2. **Управление приемами** - запись, отмена, подтверждение приемов к врачам
3. **Электронные истории болезни** - безопасное ведение медицинских записей
4. **Управление персоналом** - учет врачей, медсестер и других сотрудников
5. **Рецепты и назначения** - выписка лекарственных средств
6. **Экспорт данных** - выгрузка в JSON, CSV, PDF форматы
7. **Резервное копирование** - автоматическое создание резервных копий БД в облаке
8. **Система ролей и доступа** - разделение прав для разных типов пользователей
9. **Аудит** - логирование всех значимых действий в системе

## Технический стек

- **Backend:** Python 3.9+, Django 4.2
- **API:** Django REST Framework
- **БД:** PostgreSQL
- **ORM:** Django ORM
- **Аутентификация:** JWT (djangorestframework-simplejwt)
- **Экспорт:** ReportLab (PDF), Python CSV
- **Резервные копии:** AWS S3 / локальное хранилище
- **Фоновые задачи:** Celery + Redis
